什么是监听器
监听器是 Servlet 规范中的另一个组件，用于监听 Web 应用中的事件，并在事件发生时执行相应的逻辑。它不直接干预请求处理流程，而是用于监控应用的生命周期和属性变化。

什么是过滤器
过滤器是 Java Servlet 规范中定义的组件，可以对请求（Request） 和响应（Response） 进行拦截和预处理/后处理。它在 Servlet 被调用之前、响应返回给客户端之前执行。

Java 序列化（Serialization）是指将 Java 对象转换为字节流的过程，而反序列化（Deserialization）则是将字节流恢复为 Java 对象的过程。这是 Java 语言提供的一种内置的对象持久化和传输机制，广泛应用于 RMI（远程方法调用）、HTTP Session 持久化、分布式缓存等场景
import java.io.*;

public class SerializationDemo {
    public static void main(String[] args) throws Exception {
        User user = new User("张三", 25, "secret");

        // 序列化：对象 -> 文件
        try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("user.ser"))) {
            oos.writeObject(user);
        }

        // 反序列化：文件 -> 对象
        try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream("user.ser"))) {
            User deserializedUser = (User) ois.readObject();
            System.out.println(deserializedUser.getName()); // 张三
            System.out.println(deserializedUser.getPassword()); // null（transient 字段）
        }
    }
